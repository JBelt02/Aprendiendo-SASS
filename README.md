# Aprendiendo-SASS

En mi opinion creo que el uso de SASS es bastante bueno ya que te permite el uso de variables en lenguajes de marca como lo es HTML para poder agilizar el uso del mismo valor varicas veces o condicion . Tambien creo que es bueno ya que ayuda a que el codigo del css este mas limpio y en los casos donde el codigo es muy largo es mas facil encontrar lo que buscas que en una pagina que no lo utilice . Tiene mas funciones que agilicizan como la posibilidad del uso de funciones y demas pero no estoy tan familiarizado con su uso .
